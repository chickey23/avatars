# Source workshop (handoff)

Non-normative.

The **Source** sub-tab under **Workshops** lists Unmet Needs filtered to **`new_source`** (or all items). **Copy Cursor handoff** puts a markdown block on the clipboard for use in Cursor or another editor.

**Implementation is outside the app** (connectors, tools, prompts). The handoff template lives in [`buildHandoff.ts`](../src/services/sourceWorkshop/buildHandoff.ts).

See also [WORKSHOPS.md](./WORKSHOPS.md) and [UNMET_NEEDS.md](./UNMET_NEEDS.md).
